# Nate Pittman — Portfolio Site

Personal portfolio for Nate Pittman, Senior Product Designer at Ulta Beauty.

## Stack
Plain HTML and CSS — no frameworks, no build tools.

## Structure
```
index.html       ← Home page
css/tokens.css   ← Design tokens (colors, fonts, sizes)
css/style.css    ← Global styles and section styles
assets/images/   ← Images and media
```

## Deployment
Auto-deploys to Vercel on every push to `main`.

## Workflow
Designs are created in Figma and translated to code section by section using Claude Code.
